## ✨ Key Features

### 📊 Transaction Management
* **Create:** Easily add new income or expense records.
* **Inline Edit:** Update transaction details directly on the data table for a seamless UX.
* **Safe Deletion:** Delete transaction records with built-in confirmation prompts to prevent accidental data loss.

### 🔍 Dynamic Filtering & Tracking
* **Date Filters:** Filter transactions by specific start and end dates.
* **Smart Search:** Search records by specific categories or transaction types.
* **Real-time Calculations:** Automatically calculates the global total balance and dynamically updates the total amount of the currently filtered list.

### 🏷️ Category Management
* **On-the-fly Creation:** Add new categories instantly while creating a transaction without leaving the page.
* **Smart Segregation:** Categories are cleanly separated into Income or Expense types for better organization.
